# Submission Dicoding
## Belajar Membuat Aplikasi Backend untuk Pemula

### Tutorial

- Clone this Repo
```bash
https://github.com/ricidelphy/Bookshelf-API.git
```
- cd Bookshelf-API
```bash
cd Bookshelf-API
```
- RUN npm install
```bash
npm install
```
- Start Server
```bash
npm run start-dev
```
- Default Port: 9000, you can change the default port in src/server.js
